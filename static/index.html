About:

A game where you may or may not be playing on your own, by Norgg.

Requires an up to date version of Chrome or the nightly build of Firefox.

Server in python with ws4py, pybox2d and cherrypy.

Fontend in JavaScript/jQuery, using canvas, websockets and the webkit audio API.

Known bugs:
Most of these are due to messages from the server sometimes getting lost somewhere.

  • Biggest bug is that there are sometimes invisible bullets which will randomly kill you, sorry.
  • Occasionally the death sound doesn't play correctly.
  • Occasionally a movement direction will get stuck briefly, pressing the opposite direction should fix this.
Please report anything else at the github page.
